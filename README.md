# 🥗 API - Delivery de Alimentos Saudáveis

## 📖 Sobre o Projeto
Esta é a API RESTful de uma plataforma de delivery focada exclusivamente na venda e distribuição de alimentos saudáveis. O sistema foi projetado para gerenciar clientes, restaurantes, cardápios nutritivos e o fluxo completo de pedidos, garantindo uma integração fluida e escalável.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Java
* **Framework:** Spring Boot
* **Persistência de Dados:** Spring Data JPA / Hibernate
* **Testes de Requisições:** Insomnia
* **Versionamento de Código:** Git e GitHub

## 👥 Papéis e Responsabilidades da Equipe
Para garantir a qualidade e a entrega contínua, nossa equipe opera com uma divisão clara de tarefas:

* **Product Owner (PO):** Responsável pelo planejamento, criação do backlog, delegação de tarefas, gestão do tempo da sprint e resolução de impedimentos técnicos ou de infraestrutura da equipe.
* **Master:** Responsável pela administração do repositório, criação das *branches* para novas tarefas e execução de *merges* seguros para a *branch main* (produção).
* **Desenvolvedores (Devs):** Encarregados da codificação das funcionalidades, criação das entidades, interfaces de repositório e controladores REST (métodos GET, POST, PUT, DELETE).
* **Tester:** Especialista em qualidade que valida todas as requisições da API através do Insomnia, garantindo que as regras de negócio e os status HTTP estão retornando conforme o esperado antes da integração do código.

## 🔄 Fluxo de Trabalho (Workflow)
Nosso processo de desenvolvimento segue etapas rigorosas de validação:
1.  **Planejamento:** O PO define a funcionalidade e delega a tarefa.
2.  **Versionamento:** O Master cria uma *branch* específica para a task no GitHub.
3.  **Desenvolvimento:** Os Devs codificam a solução e realizam o *commit*.
4.  **Validação:** O Tester recebe a versão, roda o projeto e realiza os testes de endpoint no Insomnia.
5.  **Integração:** Após o "Sinal Verde" do Tester informando que o código não contém bugs, o Master realiza o *Merge* para a *branch main*.
